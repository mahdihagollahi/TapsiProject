# TapsiProject

> Demo landing of tapsi site

## Installation

> First, clone the project

`````bash
  git clone https://github.com/mahdihagollahi/TapsiProject.git
`````
> to run TapsiProject with npm

`````bash
  cd TapsiProject
  npm install
  npm run dev
`````
